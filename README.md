\# CampusHub - Student Club Management System



CampusHub is a responsive university club management website developed using HTML, CSS, Bootstrap and JavaScript.



\## Features



\- Responsive Bootstrap design

\- Student registration form

\- JavaScript form validation

\- Student search and filtering

\- Club detail modals

\- Activity registration modals

\- Dashboard homepage

\- Multi-page navigation system



\## Technologies Used



\- HTML5

\- CSS3

\- Bootstrap 5

\- JavaScript



\## Pages



\- Home

\- Clubs

\- Students

\- Activities

\- Register
